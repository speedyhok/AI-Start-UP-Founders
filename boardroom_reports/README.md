# Boardroom Reports

This directory stores the individual markdown reports generated by each co-founder agent during the boardroom session:
1. `1_idea_validator.md` - VC investment memo, TAM/SAM/SOM, KPIs, and SWOT.
2. `2_cto.md` - Technical stack, offline-first sync architecture, and MVP timeline.
3. `3_cfo.md` - Operating expenses, itemized monthly burn table, and break-even math.
4. `4_cmo.md` - Detailed target customer persona and acquisition channels.
5. `5_venture_critic.md` - Stress-testing audit, fatal vulnerabilities, and corrections.
6. `6_pitcher_blueprint.md` - Final compiled pitch and response document.
